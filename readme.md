# Sistema de Gestión de Datos de Países

## Descripción
Sistema desarrollado en Python para gestionar información sobre países, permitiendo realizar operaciones de filtrado, ordenamiento y análisis estadístico.

## Funcionalidades
- ✅ Agregar nuevos países
- ✅ Actualizar datos existentes
- ✅ Búsqueda por nombre
- ✅ Filtrado por continente, población y superficie
- ✅ Ordenamiento por diferentes criterios
- ✅ Estadísticas completas
- ✅ Persistencia de datos en CSV

## Estructuras de Datos Utilizadas
- **Listas**: Para almacenar la colección de países
- **Diccionarios**: Para representar cada país con sus atributos
- **Funciones**: Modularización del código
- **Archivos CSV**: Para persistencia de datos

## Instrucciones de Uso
1. Ejecutar el archivo `gestion_paises.py`
2. Seguir las opciones del menú interactivo
3. Los datos se cargan/guardan automáticamente en `paises.csv`

## Ejemplos de Uso
- Agregar país: Opción 1 del menú
- Buscar: Opción 3 (coincidencia parcial)
- Estadísticas: Opción 6 (promedios, máximos, mínimos)
